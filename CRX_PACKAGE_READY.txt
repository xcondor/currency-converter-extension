📝 发布 .crx 文件的步骤：

1. 构建项目
   npm run build

2. 在 Chrome 中打包
   - 访问 chrome://extensions/
   - 点击"打包扩展程序"
   - 选择 dist 目录
   - 生成 .crx 和 .pem 文件

3. 重命名文件
   mv dist.crx currency-converter.crx
   mv dist.pem currency-converter.pem

4. 保存私钥（重要！）
   - 将 .pem 文件保存到安全位置
   - 不要提交到 Git

5. 创建 GitHub Release
   - 上传 currency-converter.crx
   - 用户可以直接下载安装

⚠️ 重要提示：
- .pem 私钥文件必须妥善保管
- 更新扩展时需要使用相同的私钥
- 不要将私钥提交到 Git 仓库
